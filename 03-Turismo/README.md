# Dataset de turismo

Datos para la actividad de turismo.

En esta actividad vais a trabajar con un conjunto de datos relacionado con el turismo. El objetivo será analizar la información disponible para descubrir patrones, comparar resultados y extraer conclusiones sobre la actividad turística.

Vuestro trabajo consistirá en transformar los datos en gráficos útiles que permitan responder preguntas como: qué zonas reciben más turistas, cómo evoluciona el turismo en el tiempo o qué diferencias existen entre distintos destinos.
