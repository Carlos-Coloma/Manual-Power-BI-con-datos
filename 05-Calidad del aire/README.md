# Dataset de calidad del aire

Datos para la actividad de calidad del aire.

En esta actividad vais a trabajar con un conjunto de datos relacionado con la calidad del aire. El objetivo será analizar diferentes contaminantes atmosféricos para comprender cómo varían a lo largo del tiempo y qué información nos aportan sobre el estado del aire que respiramos.

Vuestro trabajo consistirá en crear visualizaciones que permitan responder preguntas como: qué contaminante presenta valores más altos, en qué momentos se registran peores niveles de calidad del aire o si existen patrones diarios, mensuales o estacionales.
