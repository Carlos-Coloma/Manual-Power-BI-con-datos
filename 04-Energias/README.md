# Dataset de energías

Datos para la actividad de energías.

En esta actividad vais a trabajar con un conjunto de datos relacionado con el consumo de energía en España. El objetivo será analizar cómo ha evolucionado el consumo energético a lo largo del tiempo y comparar el peso de distintas fuentes o tipos de energía.

Vuestro trabajo consistirá en crear visualizaciones que permitan responder preguntas como: qué tipo de energía se consume más, cómo cambia el consumo con los años o si existen diferencias importantes entre unas fuentes energéticas y otras.
