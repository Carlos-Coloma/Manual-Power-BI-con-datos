# Dataset de puntos de recarga de vehículos eléctricos

Datos para la actividad de puntos de recarga de vehículos eléctricos.

En esta actividad vais a trabajar con un conjunto de datos relacionado con los puntos de recarga de vehículos eléctricos. El objetivo será analizar cómo se distribuyen estos puntos, qué características tienen y si su ubicación puede responder a las necesidades reales de movilidad sostenible.

Vuestro trabajo consistirá en crear visualizaciones que permitan responder preguntas como: dónde se concentran más puntos de recarga, qué tipo de cargadores predominan, qué zonas tienen menor cobertura o si la infraestructura disponible parece suficiente.
