# Dataset de Clima

Datos para la actividad de clima.

En esta actividad vais a trabajar con un conjunto de datos relacionado con el clima. El objetivo será analizar información meteorológica para detectar patrones, comparar variables y extraer conclusiones sobre el comportamiento del tiempo atmosférico.

Vuestro trabajo consistirá en crear visualizaciones que permitan responder preguntas como: qué meses son más cálidos, cuándo llueve más, si existen diferencias entre estaciones del año o cómo evolucionan las temperaturas a lo largo del tiempo.
